# project point

Creates a new point that is a certain distance and angle from a source point.

```sig
drawing.projectPoint(drawing.createPoint(0, 0), 0, 0)
```

## Parameters

* **source**: The source point, sprite, or tile location to project from
* **angle**: The angle to project the point at, in radians
* **distance**: The distance to project the point
