# draw tilemap

Draws a tilemap to an image at a given point.

```sig
drawing.drawTilemap(image.screenImage(), null, drawing.createPoint(0, 0))
```

## Parameters

* **target**: The image to draw the tilemap on
* **tilemap**: The tilemap data to draw on the image
* **point**: The point to draw the tilemap at
