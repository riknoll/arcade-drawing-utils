# draw thick line

Draw a line to an image with a specified thickness.

```sig
drawing.drawThickLine(image.screenImage(), drawing.createPoint(0, 0), drawing.createPoint(0, 0), 2, 0)
```

## Parameters

* **target**: The image to draw on
* **start**: The start point, sprite, or tile location of the line
* **end**: The end point, sprite, or tile location of the line
* **thickness**: The thickness of the line
* **color**: The color of the line
