# draw circle

Draw a circle to an image.

```sig
drawing.drawCircle(image.screenImage(), drawing.DrawMode.Outline, drawing.createPoint(0, 0), 0, 0)
```

## Parameters

* **target**: The target image to draw on
* **mode**: The draw mode (outline or fill)
* **center**: The center point, sprite, or tile location of the circle
* **radius**: The radius of the circle
* **color**: The color of the circle
