# draw at camera origin

Draws a tilemap to the screen at the camera's origin. This is useful for drawing tilemaps directly on top of the current tilemap that scroll the same way. For example, creating a tilemap with multiple layers.

```sig
drawing.drawAtCameraOrigin(null)
```

## Parameters

* **tilemap**: The tilemap data to draw on the screen

```package
arcade-drawing-utils=github:riknoll/arcade-drawing-utils
```
