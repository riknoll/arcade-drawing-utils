# distance

Finds the distance between two points, sprites, or tile locations.

```sig
drawing.distance(drawing.createPoint(0, 0), drawing.createPoint(0, 0))
```

## Parameters

* **pointA**: The first point, sprite, or tile location
* **pointB**: The second point, sprite, or tile location
