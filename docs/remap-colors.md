# remap colors

Remaps the colors in an image. All colors in the from array will be change to the corresponding color in the to array. The from and to arrays MUST have the same length.

```sig
drawing.remapColors(myImage, drawing.colorArray(0), drawing.colorArray(0))
```

## Parameters

* **image**: The image to remap colors in
* **from**: The colors to remap from
* **to**: The colors to remap to
