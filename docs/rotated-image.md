# rotated image

Rotates an image by a certain number of 90 degree clockwise rotations. For example, a value of 1 will rotate the image 90 degrees clockwise, a value of 2 will rotate the image 180 degrees, and so on. Values less than 0 or greater than 3 will be wrapped around using modulo arithmetic.

```sig
drawing.rotatedImage(img`.`, 0)
```

## Parameters

* **image**: the image to rotate
* **rotations**: the number of 90 degree clockwise rotations
