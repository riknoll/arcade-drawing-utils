# draw line

Draw a line to an image.

```sig
drawing.drawLine(image.screenImage(), drawing.createPoint(0, 0), drawing.createPoint(0, 0), 0)
```

## Parameters

* **target**: The image to draw on
* **start**: The start point, sprite, or tile location of the line
* **end**: The end point, sprite, or tile location of the line
* **color**: The color of the line
