# scroll image

Scrolls an image by a certain amount in the x and y direction. Pixels that roll off the edge will disappear.

```sig
drawing.scrollImage(img`.`, 0, 0)
```

## Parameters

* **image**: the image to scroll
* **x**: the amount to scroll in the x direction, positive values scroll to the right and negative values scroll to the left
* **y**: the amount to scroll in the y direction, positive values scroll down and negative values scroll up
