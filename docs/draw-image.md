# draw image

Draws an image onto another image.

```sig
drawing.drawImage(image.screenImage(), img`.`, drawing.createPoint(0, 0))
```

## Parameters

* **target**: The image to draw on
* **image**: The image to draw
* **location**: The point, sprite, or tile location to draw the image at
