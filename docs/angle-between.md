# angle between

Finds the angle from one point, sprite, or tile location to another.

```sig
drawing.angleBetween(drawing.createPoint(0, 0), drawing.createPoint(0, 0))
```

## Parameters

* **pointA**: The start point, sprite, or tile location
* **pointB**: The end point, sprite, or tile location
