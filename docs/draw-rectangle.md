# draw rectangle

Draw a rectangle to an image.

```sig
drawing.drawRectangle(image.screenImage(), drawing.DrawMode.Outline, drawing.createPoint(0, 0), 0, 0, 0)
```

## Parameters

* **target**: The target image to draw on
* **mode**: The draw mode (outline or fill)
* **topLeft**: The top-left point, sprite, or tile location of the rectangle
* **width**: The width of the rectangle
* **height**: The height of the rectangle
* **color**: The color of the rectangle
